# HumanWrites 파일럿 검증 시나리오 가이드

## 개요
- **목적**: Layer 1 키스트로크 분석 임계값 교정
- **대상**: 10명 (다양한 타이핑 속도/습관)
- **소요 시간**: 참가자당 약 30-45분

## 시나리오

### (a) 직접 작성
- **설명**: 주어진 주제로 자유롭게 5분간 글 작성
- **주제 예시**: "최근 읽은 책/영화에 대한 감상", "좋아하는 계절과 그 이유"
- **예상 결과**: 자연스러운 타이핑 패턴 (기준선)
  - 다양한 WPM 변동, 자연스러운 일시정지, 수정 패턴

### (b) AI 보고 타이핑
- **설명**: ChatGPT/Claude가 생성한 텍스트를 보면서 직접 타이핑 (복사 금지)
- **방법**: AI가 생성한 200-300자 텍스트를 별도 화면에 띄우고 보면서 타이핑
- **예상 결과**: Layer 1만으로는 시나리오 (a)와 구분 어려울 수 있음
  - 비교적 일정한 WPM, 수정 빈도 낮음, 일시정지 패턴 상이

### (c) 복사-붙여넣기
- **설명**: 외부 텍스트를 복사하여 에디터에 붙여넣기
- **예상 결과**: 명확한 비인간 패턴
  - 매우 높은 "WPM", paste 이벤트 비율 > 90%, 키스트로크 거의 없음

## 데이터 수집
1. 각 참가자에게 3개 시나리오를 순서대로 수행하도록 안내
2. 각 시나리오 사이 1분 휴식
3. IndexedDB에서 세션 데이터 내보내기 (내보내기 버튼 사용)
4. 내보낸 JSON 파일을 `scripts/pilot/data/` 디렉토리에 저장
